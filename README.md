## หลักการ (ทำไมได้เป็นพัน)

Google Maps ให้ผลสูงสุด ~100 รายการต่อ 1 คำค้น ดังนั้น:

1. **Fan-out** — แตกคำค้นตามพื้นที่: `ร้านขายยา บางนา`, `ร้านขายยา ลาดพร้าว`, … × 50 เขต = หลายพันรายการ
2. **Dedup** — ร้านเดียวกันโผล่หลายคำค้น → ตัดซ้ำด้วย place_id
3. **Email** — ดึงเพิ่มจากเว็บของแต่ละร้านได้ แต่ช้าลงมาก → เปิดเฉพาะตอนต้องการ

## ติดตั้ง (ครั้งเดียว)

1. ติดตั้ง [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Windows/Mac ได้หมด)
2. ติดตั้ง Python 3 (Mac มีอยู่แล้ว / Windows โหลดจาก python.org)
3. โหลดโฟลเดอร์นี้มาไว้ในเครื่อง

## ใช้งาน

```bash
# ร้านขายยาทั่ว กทม. (50 เขต) ≈ 2,500 ราย ใช้เวลา ~3-4 ชม. (รันทิ้งไว้ได้)
python3 leads.py "ร้านขายยา" --area bkk
# (Windows ใช้ python แทน python3)

# ทั่วประเทศ 77 จังหวัด
python3 leads.py "โรงงานอาหาร" --area provinces

# เจาะพื้นที่เอง
python3 leads.py "คลินิกความงาม" --area "บางนา,อโศก,ทองหล่อ,สยาม"

# เอาอีเมลด้วย (ช้าลง 5-10 เท่า — เหมาะกับ list ที่คัดแล้ว)
python3 leads.py "โรงแรม" --area "ภูเก็ต" --email
```

ผลออกเป็นไฟล์ `<คำค้น>-<วันที่>.csv` เปิดใน Excel/Google Sheets ได้เลย — 36 คอลัมน์ (ชื่อ, เบอร์, เว็บ, อีเมล, ที่อยู่, พิกัด, เรตติ้ง, จำนวนรีวิว, เวลาเปิด ฯลฯ)

## ไอเดียใช้กับธุรกิจ B2B

| ธุรกิจคุณ | คำค้นหา lead |
|---|---|
| ขายส่งยา/เวชภัณฑ์ | ร้านขายยา, คลินิก |
| Solar cell | โรงงาน, โกดังสินค้า |
| วัตถุดิบอาหาร | ร้านอาหาร, คาเฟ่, โรงแรม |
| ซอฟต์แวร์/POS | ร้านค้าปลีก, ร้านทอง, อู่ซ่อมรถ |
| ประกัน/การเงิน | บริษัท, หจก, สำนักงานบัญชี |

**ขั้นต่อ:** เอา CSV เข้า Google Sheets → กรองเรตติ้ง/จำนวนรีวิว → ทีมเซลล์โทร หรือส่งอีเมลแนะนำตัว

## จะโดน Google แบนไหม?

ไม่มี "แบนถาวร" — ที่เกิดได้คือ IP โดน rate limit ชั่วคราว (CAPTCHA ไม่กี่ชั่วโมงก็หาย) และเครื่องมือไม่ได้ login บัญชี Google ใดๆ บัญชีพี่จึงไม่เสี่ยง กติกาให้ปลอดภัย:

- งานใหญ่รันกลางคืน ทิ้งช่วงระหว่างแคมเปญ — อย่ารันวนทั้งวันติดกันหลายวัน
- ระดับที่ทดสอบแล้วปลอดภัย: 50 คำค้น / ~2,500 ราย ต่อรอบ จาก IP บ้าน
- ยิงหนักกว่านั้นมาก → ใส่ proxy (`-proxies` flag ของ scraper) หรือใช้ Apify (ข้างล่าง)
- **ฝั่งที่เสี่ยงจริงคือการใช้ข้อมูล:** โทร/อีเมลแนะนำธุรกิจแบบ B2B ได้ แต่อย่า spam หว่าน (PDPA)
- ข้อมูลถูกต้อง ณ วันที่ดึง — Google Maps เปลี่ยนตลอด ควรดึงใหม่ก่อนใช้จริง
