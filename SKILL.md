# gmaps-leads

ดึงรายชื่อธุรกิจจาก Google Maps → CSV ด้วย `leads.py` ในโฟลเดอร์นี้

## วิธีทำ

1. ถามผู้ใช้แค่ 2 อย่าง (ถ้ายังไม่บอก): **ประเภทธุรกิจ** + **พื้นที่**
2. เลือก area:
   - ทั่ว กทม. → `--area bkk` (50 เขต)
   - ทั่วประเทศ → `--area provinces` (77 จังหวัด)
   - เจาะจง → `--area "พื้นที่1,พื้นที่2"` (ชื่อเขต/อำเภอ/จังหวัด/ย่าน)
3. รัน (background เสมอถ้าคำค้นเยอะ — วัดจริง: 50 คำค้น depth 5 ≈ 3-4 ชม. ได้ ~2,500 ราย):
   ```bash
   python3 leads.py "<ประเภทธุรกิจ>" --area <พื้นที่> --depth 5
   ```
   - เครื่องที่มี binary: นำหน้าด้วย `GMAPS_BIN=/path/to/google-maps-scraper`
   - ไม่มี binary → สคริปต์ใช้ Docker เอง (image `gosom/google-maps-scraper`)
   - ผู้ใช้ขออีเมลด้วย → เพิ่ม `--email` และเตือนว่าช้าลง 5-10 เท่า
4. เสร็จแล้วรายงาน: จำนวนหลัง dedup, มีเบอร์/เว็บ/อีเมลกี่ราย, path ไฟล์ CSV เป็น clickable link

## Troubleshooting

- **ไม่มีทั้ง Docker และ binary** → ให้ผู้ใช้ลง Docker Desktop แล้วรันใหม่
- **ได้ 0 ราย** → คำค้นแคบไป ลองคำกว้างขึ้น (เช่น "คลินิก" แทน "คลินิกกายภาพเฉพาะทาง")
- **โดน rate limit / ค้างนาน** → ลด `-c` ใน leads.py หรือแบ่งรันทีละภาค
